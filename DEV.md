# WeedTrust Development Notes

## Requirements
- Java 17
- NodeJS 22
- Composer 2.7.7
- PHP 8.3.10
- XCode
- Android Studio

## Commands
### Client
#### Local Development
<code>ionic serve</code>
#### iOS Development
<code>npx cap run ios</code>
#### Android Development
<code>npx cap run android</code>
